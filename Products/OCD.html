Description

OCD is a game similar to Free Cell but with the look
and feel of Bejewelled. The object of the game is to line up all
of the dots in their respective columns. From left to right the columns
are Blue, Green, Red, Orange, and Purple.

Game Play

There are three difficulty modes with at least one more
planned. In easy mode, you can move any dot to any location up,
down, left, or right. In normal mode, when a column has all of the
dots for its color in place, the column is locked and the dots in that
column can no longer be moved. In hard mode, not only do the columns lock
but the row locks too if one dot of each color are lined up in the
column color order.

Scoring

Scoring adds an additional level of strategy. Each move made is
minus 1 point unless either dot is placed on the same column color
as the dot, which gives 5 points. This allows for greater strategy when
moving dots across the board.

Known Issues
- Game takes longer than needed to load after selecting difficulty
- Game does not respond if a new game is started
from the main menu after completing a game
